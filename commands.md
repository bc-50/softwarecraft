### Git commands

- To init
```git init```
- To check status
```git status```
- To stage a file
```git add main.go```
- To commit a change
```git commit -m "Add main.go file```
- In general commit messages are written in simple present tense
- To check current branch. * indicates the current branch
```git branch```
- To unstage a file
```git restore --staged commands.md```
